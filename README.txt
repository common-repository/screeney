Connects your website with the Screeney bug tracking web application.

== Description ==

Whether you're developing a new site or just looking to iron out any issues, Screeney helps you to visually capture client feedback and action it quickly.

This plugin allows you to connect your website to the Screeney bug tracking application so there's no longer a need to add code to your website.

>**[Visit the Screeney website to get an account](https://screeney.com/)**

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `screeney` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
